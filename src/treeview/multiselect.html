The TreeView component allows you to select multiple nodes by enabling the allowMultiSelection property.

In this demo, the TreeView was enabled with multiple selection, so you can select multiple nodes by pressing:

  • CTRL key - you can press the CTRL key and select your desired nodes.
  • SHIFT key - select any node and with pressing shift key select another node, which selects all the nodes in-between that nodes

In case of Mobile device,

  • Select a node and you can see a small popup to enable multi selection, after pressed that popup now you are allowed to select multiple nodes

For more information, you can refer the Multi Selection section from the documentation.
