All the Gantt actions can be controlled via keyboard keys and are availed using the allowKeyboard property, which is set to true by default. The applicable key combinations and their relative functionalities are listed below.

Gantt component features are segregated into individual feature-wise modules. To use a selection, toolbar, edit and markers inject the SelectionService , ToolbarService , EditService and DayMarkersService into the @NgModule.providers section respectively.
