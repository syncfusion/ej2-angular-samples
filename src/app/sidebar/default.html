The Sidebar sample demonstrates the default functionalities of the Sidebar. Click on the hamburger menu icon to expand/collapse the sidebar. Click the TreeView node to see the corresponding folder’s mail details.

The Sidebar is an expandable and collapsible component that typically acts as a side container to place primary or secondary content alongside the main content. It provides flexible options that can be shown and hidden based on user interactions. Any type of HTML content or component can be placed in the Sidebar for quick access and easy navigation, like quick references, menus, lists, and tree views.

In this demo, the Sidebar is populated as like webmail with folder structure.
