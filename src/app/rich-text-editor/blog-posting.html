RichTextEditor is a WYSIWYG editing control which will reduce the effort for users while trying to express their formatting word content as HTML or Markdown format. So, RichTextEditor can easily customized to use for blog posting, forums as an editor for response.

Injecting Module

RichTextEditor features are segregated into individual feature-wise modules. In this demo we have used following injectable services ToolbarService, LinkService, ImageService, HtmlEditorService, QuickToolbarService into the @NgModule.providers section.
