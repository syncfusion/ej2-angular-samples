In this example, you can see how to render and configure the Period Selector. Tooltip is enabled in this example, to see the tooltip in action, hover a point or tap on a point in touch enabled devices.


Injecting Module

Range Navigator component features are segregated into individual feature-wise modules. To use period selector, we need to inject PeriodSelectorService module into the @NgModule.providers section.
