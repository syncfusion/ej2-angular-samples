This sample visualizes stock chart with multiple panes.

In this example, you can see how to render and configure the Stock chart with volume. CandleSeries is used to represent selected data value and ColumnSeries is used to represent the volume.


Injecting Module:

Stock Chart features are segregated into individual feature-wise modules. To use Candle Series, Column Series and DateTime axis in stock chart, we need to inject ColumnSeriesService, CandleSeriesService and DateTimeService into the @NgModule.providers section.
