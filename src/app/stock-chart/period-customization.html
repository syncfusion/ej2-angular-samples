This sample visualizes data for intraday using stock chart. Period and range selector help us to navigate different of data.

In this example, you can see how to render and configure the Stock chart. LineSeries is used to represent selected data value.


Injecting Module:

Stock Chart features are segregated into individual feature-wise modules. To use Line Series and DateTime axis in stock chart, we need to inject LineSeriesService and DateTimeService into the @NgModule.providers section.
