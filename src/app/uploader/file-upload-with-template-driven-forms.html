This example demonstrates how to integrate Upload component with Angular template-driven forms.

The submit button will be enabled once the form is filled with valid values. You can fill the details and submit the template-driven form.

The Upload component can be integrating with Angular template-driven forms. In this sample, all the form fields are validated by using ngModel. The selected file's value is updated to Textbox using selected event of Upload component. The form will be reset to defaults after submitting the form.

The following validation is enabled in this example:

  • Name - required
  • Email - required and valid email address
  • Mobile number - required, number only, and minimum length as 10
  • Photo - allow to upload images
  • Address - optional

For more information about Angular template-driven forms can be found in this reference link.
