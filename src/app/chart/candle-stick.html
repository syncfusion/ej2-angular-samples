This sample visualizes the AAPL historical data with default candle series in the chart. Tooltip and crosshair shows the information about the data and period.

AAPL Historical

In this example, you can see how to render and configure the candle type chart. Candle type chart is used to represent the price movements in stock. You can use border, fill properities to customize the data appearance.


Injecting Module

Chart component features are segregated into individual feature-wise modules. To use Candle feature, we need to inject CandleService into the @NgModule.providers section.

More information on the Candle series can be found in this documentation section.
