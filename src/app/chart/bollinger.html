This sample illustrates a stock chart with candle series and a Bollinger band indicator. Trackball shows the information about the stock, signalline, upperline, and lowerline value of a day.

In this example, you can see how to render and configure the BollingerBands. BollingerBands are used to represent the price movements in stock.


Injecting Module

Chart component features are segregated into individual feature-wise modules. To use BollingerBands, we need to inject BollingerBandsService into the @NgModule.providers section.

More information on the BollingerBands can be found in this documentation section.
