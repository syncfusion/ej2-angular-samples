This sample demonstrates polar series with scatter type for GDP for different countries in recent years. The switching between polar and radar series can be done by using Series Type in property panel.

Series Type:

In this example, you can see how to render and configure the polar series with scatter type charts.

Tooltip is enabled in this example, to see the tooltip in action, hover a point or tap on a point in touch enabled devices.


Injecting Module

Chart component features are segregated into individual feature-wise modules. To use Polar series with drawType scatter feature, we need to inject PolarSeriesService and ScatterSeriesService into the @NgModule.providers section.

More information on the polar scatter series can be found in this documentation section.
