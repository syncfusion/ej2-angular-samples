In this example, you can see how to render and configure the recurrence stripline in chart. Striplines are used to annotate the highlighted region with some notes. You can use border, fill dash-array properties to customize the striplines.

Tooltip is enabled in this example, to see the tooltip in action, hover a point or tap on a point in touch enabled devices.


Injecting Module

Chart component features are segregated into individual feature-wise modules. To use column series, we need to inject Stripline module into the @NgModule.providers section.

More information on the stripline recurrence can be found in this documentation section .
