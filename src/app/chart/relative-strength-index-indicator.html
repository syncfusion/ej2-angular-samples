This sample illustrates a stock chart with candle series and a Relative Strength Index indicator. Trackball shows the information about the stock, signalline, lowerline, and upperline value of a day.

In this example, you can see how to render and configure the RSI Indicator. RSI Indicator is used to represent the price movements in stock.


Injecting Module

Chart component features are segregated into individual feature-wise modules. To use RSI Indicator, we need to inject RsiIndicatorService into the @NgModule.providers section.

More information on the RSI Indicator can be found in this documentation section.
