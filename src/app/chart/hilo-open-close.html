This sample visualizes the AAPL historical data with default HILO Open Close series in the chart. Tooltip and crosshair shows the information about the data and period.

AAPL Historical

In this example, you can see how to render and configure the Hilo Open Close type chart. HiloOpenClose type chart is used to represent the price movements in stock. You can use border, fill properities to customize the data appearance.


Injecting Module

Chart component features are segregated into individual feature-wise modules. To use Hilo Open Close feature, we need to inject HiloOpenCloseSeriesService into the @NgModule.providers section.

More information on the Hilo Open Close series can be found in this documentation section.
