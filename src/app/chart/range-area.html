This sample visualizes the maximum and minimum temperatures of different months with default range area series in the chart. Zoom the chart to check the temperature for week or day.

In this example, you can see how to render and configure the range area type charts. You can use border, fill properities to customize the data appearance. dataLabel is used to represent individual data and its value.


Injecting Module

Chart component features are segregated into individual feature-wise modules. To use Range area series feature, we need to inject RangeAreaSeriesService into the @NgModule.providers section.

More information on the area types series can be found in this documentation section.
