This sample demonstrates polar series with stacking column type. The switching between polar and radar series can be done by using Series Type in property panel.

Properties
Series Type:

In this example, you can see how to render and configure the polar series with StackingColumn draw type charts. You can use border, fill properties to customize the data appearance. dataLabel is used to represent individual data and its value.

Tooltip is enabled in this example, to see the tooltip in action, hover a point or tap on a point in touch enabled devices.


Injecting Module

Chart component features are segregated into individual feature-wise modules. To use column feature, we need to inject PolarSeriesService into the @NgModule.providers section.

More information on the polar column types series can be found in this documentation section.
