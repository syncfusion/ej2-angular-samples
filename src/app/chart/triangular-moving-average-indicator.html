This sample illustrates a stock chart with candle series and a Triangle Moving Average indicator. Trackball shows the information about the stock and signal value of a day.

In this example, you can see how to render and configure the TMA Indicator. TMA Indicator is used to represent the price movements in stock.


Injecting Module

Chart component features are segregated into individual feature-wise modules. To use TMA Indicator, we need to inject TmaIndicatorService into the @NgModule.providers section.

More information on the TMA Indicator can be found in this documentation section.
