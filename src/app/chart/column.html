This Angular column chart example visualizes the production of Walnuts and Almonds for different countries in 2023.

In this example, you can see how to render and configure a column chart. The column chart is used to compare the frequency, count, total, or average of data in different categories.

Tooltip is enabled in this example. To see the tooltip in action, hover over a point or tap on a point in touch-enabled devices.

Injecting Module

Chart component features are segregated into individual feature-wise modules. To use column feature, we need to inject ColumnService into the @NgModule.providers section.

More information about the column series can be found in this documentation section.
