This sample illustrates a rounded column series. Data points values are showed by using data label.

In this example, you can see how to render and configure the column type charts with rounded corner.

Tooltip is enabled in this example, to see the tooltip in action, hover a point or tap on a point in touch enabled devices.


Injecting Module

Chart component features are segregated into individual feature-wise modules. To use column feature, we need to inject ColumnSeriesService into the @NgModule.providers section.

More information on the column types series can be found in this documentation section.
