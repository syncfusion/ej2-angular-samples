This Angular Spline Area Chart example visualizes the inflation rate comparision for two countries by using spline area series.

In this example, you can see how to render and configure the spline area chart. This chart plots a fitted curve through each data point in a series. It is used to represent time-dependent data and show trends in data at equal intervals.


Injecting Module

Chart component features are segregated into individual feature-wise modules. To use spline area series feature, we need to inject SplineAreaService into the @NgModule.providers section.

More information about the spline area series can be found in this documentation section.
