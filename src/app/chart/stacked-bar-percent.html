This sample visualizes percentage of sales comparison for different fruits with default 100% stacked bar series in chart. Legend in the sample shows the information about the series.

In this example, you can see how to render and configure the 100% stacked bar type charts. Stacks the points in the series horizontally and also you can use stackingGroup property to group the stacking collection based on categories. You can use border, fill properties to customize the vertical bar. dataLabel is used to represent individual data and its value.

Tooltip is enabled in this example, to see the tooltip in action, hover a point or tap on a point in touch enabled devices.


Injecting Module

Chart component features are segregated into individual feature-wise modules. To use 100% stacked bar series, we need to inject StackingBarSeriesService into the @NgModule.providers section.

More information on the bar type series can be found in this documentation section.
