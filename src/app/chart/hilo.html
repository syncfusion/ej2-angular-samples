This sample visualizes the AAPL historical data with default HILO series in the chart. Tooltip and crosshair shows the information about the data and period.

In this example, you can see how to render and configure the Hilo type chart. Hilo type chart is used to represent the price movements in stock. You can use border, fill properities to customize the data appearance.


Injecting Module

Chart component features are segregated into individual feature-wise modules. To use Hilo feature, we need to inject HiloSeriesService into the @NgModule.providers section.

More information on the column series can be found in this documentation section.
