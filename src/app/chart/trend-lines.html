This sample visualizes the trend of Indian rupees and US dollar variation with trendline in the chart. The type of trend line can be changed by using TrendLine Type, forward and backward forecasting of trendlines can be changed by Forward Forecasting and Backward Forecast respectively. Polynomial and period for a trendlines can be changed by using Polynomial order and Period.

In this example, you can see how to render and configure the Trend Lines. Trend Lines is used to represent the price movements in stock. You can use border, fill properities to customize the data appearance.


Injecting Module

Chart component features are segregated into individual feature-wise modules. To use Trend Lines, we need to inject TrendlinesService into the @NgModule.providers section.
