This sample demonstrates customization axes in linear gauge. You can change minimium and maximum value of pointer by using Axis Minimum and Axis Maximum in property panel. Inversed and opposed position of an axis can be changed by Axis Inversed and Axis Opposed. Type of pointer and placing marker can be customized by Pointer Type and Marker Placement.
