Registration Form

* Enter your username * Enter a valid username
* Enter your password * Enter a valid password
* Enter your email address * Enter a valid email address
* Enter your mobile number * Enter a valid mobile number
*This button is not a submit type and the form submit handled from externally.

This example demonstrates how to design template-driven form using Textbox component. The submit button will be enabled after filling the form with valid values. You can fill the details and submit the form.

The Textbox component is integrated with template-driven forms and it allows you to bind the value to model using ngModel. The textbox appearance will be changed based on its success or failure state. After the form is submitted, the values will be reset to defaults. The following validation is enabled in this example:

  • First name and Last name - required
  • Username and password - required and minimum length as 6
  • Email - required and valid email address
  • Mobile number - required, number only, and minimum length as 10
