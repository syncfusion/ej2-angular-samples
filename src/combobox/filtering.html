The ComboBox has built-in support to filter the data source when allowFiltering enabled. And it performs when type characters in the search box. In filtering event, you can filter down the data source and return the resulted data to ComboBox via updateData method to display its list items.

In this sample illustrates that, query the data source and passed the resulted data to ComboBox through the updateData method in filtering event.

More information on the filtering feature configuration can be found in this documentation section.
